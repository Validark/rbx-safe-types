copyWithin

  • copyWithin(target: number, start?: undefined | number, end?: undefined | number): this

  • Inherited from Array.copyWithin

    • Defined in es.d.ts:473

    Shallow copies part of an array to another location in the same array and returns it, without modifying its size.

    Parameters

    • target: number

      Zero based index at which to copy the sequence to. If negative, target will be counted from the end. If target is at or greater than arr.length, nothing will be copied. If target is positioned after start, the copied sequence will be trimmed to fit arr.length.

    • Optional start: undefined | number

      Zero based index at which to start copying elements from. If negative, start will be counted from the end. If start is omitted, copyWithin will copy from the start (defaults to 0).

    • Optional end: undefined | number

      Zero based index at which to end copying elements from. copyWithin copies up to but not including end. If negative, end will be counted from the end. If end is omitted, copyWithin will copy until the end (default to arr.length).

    Returns this

sort

  • sort(compareFunction?: undefined | function): Array<string>

  • Inherited from ReadonlyArray.sort

    • Defined in es.d.ts:422

    The sort() method copies the elements of an array and returns a sorted array. The default sort order is built upon converting the elements into strings, then comparing via the < operator.

    Parameters

    • Optional compareFunction: undefined | function

      Specifies a function that defines the sort order. If omitted, the array is sorted according to each character's Unicode code point value, according to the string conversion of each element. (a, b) => a - b is a good starting point for numbers. If compareFunction(a, b) is less than 0, sort a to an index lower than b (i.e. a comes first), and vice versa.

    Returns Array<string>
